# --- cowboy 2.8.0 / cowlib 2.9.1 advisories, fixed in 2.15.0 / 2.16.1 ---

# never invoked. Versions are pinned by the grpc library (cowlib ~> 2.9.0 override in mix.exs);
# bumping requires a grpc upgrade. Real fix tracked for the dep-bump branch.
