# secrethub dependency-audit suppressions.

# secrethub's only HTTP surface is the OIDC discovery Plug.Router (GET/JSON) behind cluster ingress;
# it also uses gRPC (grpc_gun) and the Ecto stack. cowlib/cowboy come via grpc_gun/cowboy, hackney
# via httpoison, postgrex/decimal via Ecto. Every advisory below is cleared only by an out-of-scope
# stack bump tracked for the dep-bump branch (same handling as public-api/v2).
