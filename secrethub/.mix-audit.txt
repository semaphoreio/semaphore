# version released upstream as of this bump; cowlib <= 2.16.1 is affected).
# Secrethub does not generate Set-Cookie headers from user input — it exposes a
# gRPC API plus a small internal Plug.Cowboy plane that serves only static OIDC
# JWKS/discovery JSON, so cow_cookie:cookie/1 is never reached with user data.

# ecto/ecto_sql version bump that is out of scope for a security patch PR.
